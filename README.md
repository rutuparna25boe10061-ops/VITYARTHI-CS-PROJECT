📌 Project Title
 Calculator App ( Python Tkinter)

📖 Overview
This project is a simple GUI-based calculator built using Python’s Tkinter library. It performs basic arithmetic operations like addition, subtraction, multiplication, and division, along with extra features such as square and square root.
The application is user-friendly and designed with a clean interface, making it suitable for beginners learning GUI development in Python.
________________________________________
________________________________________
✨ Features
Basic Features
•	Addition, subtraction, multiplication, division
•	Square functions (x^2)
•	Square root function
•	Decimal input
•	Keyboard input support
•	Backspace & clear (C) option
User Interface
•	Tkinter-based GUI
•	Responsive layout using grid
